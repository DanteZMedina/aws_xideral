Tecnologías

Proyecto Final - NYC Taxi Data Dashboard

Desarrollo de una aplicación desplegada en AWS utilizando Python, Spark y Lambda para procesar datos de los taxis de nueva york, con el objetivo de analizar patrones de viaje, tarifas, demanda y otros aspectos relacionados con el transporte urbano. La aplicación se desplegará en AWS EC2, utilizando lambda para procesar los datos y generar visualizaciones interactivas que permitan a los usuarios explorar la información de manera intuitiva.
